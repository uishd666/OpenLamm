## <span style="font-size: 1.8rem; font-weight: bold; color: #333;">ChEF Benchmarking Dataset</span>

<br />

#### <span style="font-size: 1.3rem; font-weight: bold; color: #333;">Omnibenchmark</span>

<br />

<p>Download <a href="https://entuedu-my.sharepoint.com/:f:/g/personal/yuanhan002_e_ntu_edu_sg/El2wmbzutJBOlu8Tz9HyDJABMmFtsG_8mq7uGh4Q7F1QSQ?e=NyroDS" target="_blank" style="color: #007bff; text-decoration: none; font-weight: bold;">Omnibenchmark</a> for fine-grained classification dataset and <a href="https://github.com/ZhangYuanhan-AI/Bamboo" target="_blank" style="color: #007bff; text-decoration: none; font-weight: bold;">Bamboo Label System</a> for hierarchical catergory labels. </p>

<br />

<p>
We sampled and labeled Omnibenchmark meticulously by using a hierarchical chain of categories, facilitated by the Bamboo label system. 
</p>

<br />

<div style="background-color: #f8f9fa; padding: 16px; border-radius: 8px; font-family: 'Courier New', Courier, monospace; font-size: 1rem;">
    <pre>
    python ChEF/data_process/Omnibenchmark.py
    </pre>
</div>

<br />

<p>
You can also directly download the labeled Omnibenchmark dataset from <a href="https://openxlab.org.cn/datasets/LAMM/ChEF" target="_blank" style="color: #007bff; text-decoration: none; font-weight: bold;">OpenXLab</a>.
</p>
